# 解剖学クイズ 519・図入り版

GoodNotes資料をもとにした、スマートフォン対応の解剖学クイズアプリです。

## 収録範囲

- 上肢：121問
- 下肢：120問
- 背部・胸部・腹部：143問
- 骨盤部・頭頸部・中枢神経系：135問
- 合計：519問

資料上、問題番号429は欠番です。内容を推測して補完せず、原資料の番号を維持しています。

## 機能

- 想起学習・四択
- 間隔反復
- 弱点優先出題
- 分野別学習
- 学習履歴を端末内に保存
- スマートフォン・オフライン対応
- 答えと一緒に、GoodNotes元資料の対応図を表示
- 図をタップして拡大表示

## 図の使い方

1. 右上の設定を開く
2. 「GoodNotesファイルを選択」を押す
3. 学習する分野の `.goodnotes` ファイルを選択する
4. 問題の答えを表示すると、その下に対応図が表示される

GoodNotesファイルはブラウザ内で解析され、外部へ送信されません。現在の版ではページを閉じた後に再度ファイルを選択する必要があります。
