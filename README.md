# 解剖学クイズ 519・図入り・英語／ラテン語版

GoodNotes資料をもとにした、スマートフォン対応の解剖学クイズアプリです。

## 収録範囲

- 上肢：121問
- 下肢：120問
- 背部・胸部・腹部：143問
- 骨盤部：42問
- 頭頸部：83問
- 中枢神経系：10問
- 合計：519問

資料上、問題番号429は欠番です。内容を推測して補完せず、原資料の番号を維持しています。

## 日本語・英語・ラテン語

各問題で答えを表示すると、日本語の資料回答の直下に次の内容を表示します。

- 英語名
- ラテン語名

複数の構造を答える問題では、英語とラテン語も項目ごとに分けて表示します。ラテン語は解剖学で一般的に用いられる名称を採用し、資料の日本語表現に対応させています。

## 分野別学習

ホームでは、まず次の大分類から選択します。

- 上肢
- 下肢
- 背部・胸部・腹部
- 骨盤部
- 頭頸部
- 中枢神経系

大分類を選ぶと、筋・関節・運動作用・血管・リンパ・神経・臓器などの小分類が表示されます。小分類ごとに問題数、定着数、復習期限の問題数を確認し、その分野だけを重点的に学習できます。

学習条件画面、問題一覧、学習記録も大分類・小分類の二段階フィルターに対応しています。

## 機能

- 想起学習・四択
- 間隔反復
- 弱点優先出題
- 大分類・小分類による重点復習
- 大分類・小分類・問題ごとの学習状況表示
- 問題一覧から未学習・学習中・定着を手動変更
- 学習履歴を端末内に保存
- スマートフォン・オフライン対応
- 日本語回答の下に英語名・ラテン語名を表示
- 答えと一緒に、GoodNotes元資料の対応図を表示
- 図をタップして拡大表示
- 抽出した図をIndexedDBへ圧縮保存

## 図の使い方

1. 右上の設定を開く
2. 「GoodNotesファイルを選択」を押す
3. 学習する分野の `.goodnotes` ファイルを選択する
4. 初回の解析・保存が完了するまで待つ
5. 以降はファイルを選び直さなくても対応図が自動表示される

GoodNotesファイルはブラウザ内で解析され、外部へ送信されません。元ファイル全体ではなく、問題に対応する図だけを圧縮して端末内に保存します。SafariのWebサイトデータを削除した場合や、端末の空き容量不足でデータが消去された場合は再読込が必要です。
